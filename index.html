Allianz Online Forms

For Allianz Global Assistance I developed a forms framework for handling insurance claims.

The frontend is build with plain Javascript(ES5) and KnockoutJS components.
On the backend .NET MVC, Web API and Entity Framework are used.

Forms are submitted directly to Allianz through their SOAP services, helping them achieve fully automated claim handling.

Gherkin test scenario's are implemented with SpecFlow for end-to-end regression testing.
